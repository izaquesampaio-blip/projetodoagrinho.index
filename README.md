# 🌾 Projeto do Agrinho

Educação agrícola interativa com tecnologia moderna integrando p5.js

## 📋 Sobre o Projeto

O **Projeto do Agrinho** é uma plataforma educativa que utiliza **p5.js** para criar visualizações e simulações interativas sobre agricultura sustentável e práticas agrícolas modernas.

Este projeto combina:
- 🎨 **Design moderno e responsivo**
- 💻 **Tecnologia web interativa**
- 📚 **Conteúdo educativo agrícola**
- 🌱 **Foco em sustentabilidade**

## 🚀 Funcionalidades

- ✅ **Navbar navegável** com menu responsivo
- ✅ **Hero section** atrativa com call-to-action
- ✅ **Seção sobre** descrevendo o projeto
- ✅ **Integração p5.js** com iframe e botões de ação
- ✅ **Seção de recursos** destacando features principais
- ✅ **Seção de contato** com links para redes
- ✅ **Footer** com informações e redes sociais
- ✅ **Design responsivo** (mobile, tablet, desktop)
- ✅ **Animações** suaves e transições
- ✅ **Acessibilidade** otimizada

## 🔧 Como Usar

1. **Clone o repositório**
```bash
git clone https://github.com/izaquesampaio-blip/projetodoagrinho.index.git
cd projetodoagrinho.index
```

2. **Abra o arquivo HTML**
```bash
# Windows
start index.html

# macOS
open index.html

# Linux
xdg-open index.html
```

3. **Ou use um servidor local (Node.js)**
```bash
npx http-server
```

Acesse `http://localhost:8080` no navegador.
